What Are Computer Graphics?

Computer graphics deals with images. The images are moved around. Their scale is changed. Three dimensions are projected onto two dimensions. All the main operations are done by matrices.
Strang, G. (2009). Computer Graphics. In Introduction to Linear Algebra (4th ed., pp. 459). Wellesley, MA: Wellesly-Cambridge Press.

History of Graphics

  • 2D graphics: really more suited for an art history class
  • 3D graphics: started in the 1960s.
  • Ivan Sutherland
    • Developed computer program called SKETCHPAD, one of the first 2D CAD programs
    • Came up with 10 problems in computer graphics - most have been solved at least partially by now (but others came up with more!)
  • 1970s: Saw first use in movies (wireframe)
  • 1980s: Real time graphics hardware mainstream for 2d games and computer animation
  • 1990s: First real time 3D graphics hardware and software started appearing. Standardized libraries started appearing (like DirectX and OpenGL). First computer animated feature length film released (Toy Story)
  • 2000s: 3D graphics hardware and software becomes regularly used. Many innovations improve performance and ease of use.
  • 2010s: Virtual reality hardware becomes mainstream, which allows users to navigate a 3D space much more intuitively.
